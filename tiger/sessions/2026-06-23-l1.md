### H1 — Draft's "digest-only" grounding ceiling (VALUE) — the #1 finding
**Every** draft value agent (noa, ingrid, kenji) independently hit the same wall: `buildDraftPrompt`
(drafting.ts:218) is fed only the **pre-scored criteria digest** (`name [status]: evidence — rationale`,
criteriaLines drafting.ts:158) + exhibits — never the applicant's raw CV / source materials. Combined
with STRICT RULE 1 ("if a detail is not provided, argue generally without fabricating", drafting.ts:225),
the letter is *pushed toward generic, conclusory prose for exactly the cases whose value is in
specifics*: Kenji's OSS adoption curve / named adopters / maintainer authority, Noa's production names +
40M streams + guild award, Ingrid's biennale venue tier. The wedge's framing is correct (draftFraming
covers O-1B-lead + EB-1A-final-merits); the **evidence ceiling is the digest**, so it saves the attorney
the scaffold but NOT the evidence-enrichment — the load-bearing part for behind-the-scenes / OSS / arts.
- Verdicts: noa CONDITIONAL, ingrid CONDITIONAL, kenji CONDITIONAL (high FAIL risk). None PASS.
- Fix: feed bounded **raw evidence** into the draft prompt (a fenced `<<<BACKGROUND_MATERIALS>>>`
  block, "enrich the assertions, never cite, invent nothing"), OR widen the qualify `evidence` field
  (criteria-text MAX_TEXT=4000) + instruct the scorer to extract adoption/production depth. Lifts the
  ceiling from digest→narrative to CV→narrative; estimated grounding 2.5/5 → 4/5.

### H2 — The premium op isn't on a premium model (MODEL/CODE) — config question, now confirmed
All three Lens-1 agents independently flagged it: `tier:"long"` → `GEMINI_DRAFT_MODEL ?? fast`
(config.ts:36-40). `GEMINI_DRAFT_MODEL` is **unset in this checkout**, so the `xl`/12-token draft and
the `heavy`/5-token RFE run on the **same `gemini-3-flash-preview`** as the 1-token `guidance` op — the
cost-tier differentiation is defeated for the two most valuable, most expensive ops.
- Fix: (a) set a real long-context model and benchmark it (Lens 3), or (b) if Flash is intentional,
  log a loud startup warning + document it and **reprice** the xl tier (you're charging 12 for Flash).
  Also: RFE uses `tier:"long"` though it's a shorter structured response — consider `tier:"fast"`.
